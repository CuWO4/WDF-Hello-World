# WDF HELLO WORLD

generates the hello world program in a very confusing way

---

## Usage

```bash
make generate
```
and then there should be a hello program in the current directory, which prints `hello, world` when executed.

## Environment

* Ubuntu 20.04.6 LTS

* clang 13.0.1

* GNU Make 4.2.1
